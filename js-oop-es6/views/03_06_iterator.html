6. for in / for of 반복문과 enumerable, iterable 속성

  • 반복문

    var 오브젝트 = {name:"Kim", age:30} for(let i = 0; i < 3; i++){ }

  • for in 반복문

    Object에 있던 값을 전부 하나씩 꺼내서 사용할 때

    enumerable한 것만 반복해준다.

    enumerable 이라는 것은 object 생성시

    Object.getOwnPropertyDescriptor(오브젝트, 'name'); 으로 데이터에 셋팅된 설정값을 확인할 수 있다.

    enumerable 이라는 것은 오브젝트["name"]에서는 셀수 있는 값이기 떄문에 true(기본값)로 설정되어있다.

    Object 자료형에만 사용

    var 오브젝트 = {name:"Kim", age:30} for(let key in 오브젝트){ console.log(오브젝트[key]); } for(let key in 오브젝트){ console.log(오브젝트["name"]); }

  • class 부모 { } 부모.prototype.name = "Park"; var 오브젝트 = new 부모(); for(var key in 오브젝트){ if(오브젝트.hasOwnProperty(key)){ console.log(오브젝트[key]); } }

  • for of 반복문

    Array, 문자, arguments, NodeList, Map, Set에 사용 가능

    이터러블한 요소에만 사용가능

    var 어레이 = [2,3,4,5]; for(var 자료 of 어레이){ console.log(자료); } for(var 자료 of 'ㄴㄻㄴㅇㅁㄴㅇ'){ console.log(자료); //ㄴㄹㅁㄴㅇㅁㄴㅇ }
