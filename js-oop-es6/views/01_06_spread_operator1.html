1. 스프레드 연산자 (Spread Operator) 활용방법 1

  • 스프레드 연산자 활용

    배열을 쉽게 합치거나 기존의 배열을 활용하여 새 배열을 만들고 싶을 때

    deep copy를 하고 싶을 때

    스프레드 연산자를 이용하여 복사하면 다른 메모리에 같은 배열을 새로 복사하게 된다.

    즉 스프레드 연산자를 이용하여 복사된 변수의 배열의 값을 변경하여도
              복사해왔던 원본배열은 영향을 받지 않는다.

    변수에 배열을 복사할 때 등호를 이용한 복사는 같은 메모리 주소를 공유할 뿐이다. (동일한 배열을 참조)

    var a = [1,2,3];
    var b = [4,5];
    var c = [...a] // [1,2,3] :a의 대괄호는 삭제되고 c의 대괄호만 남는다.
    var d = [...a, ...b] // [1,2,3,4,5] : a,b의 배열의 값만 가져와서 배열에 새로 넣는다.
    var o1 ={a:1, b:2};
    var o2 = {...o1, c:3}
    console.log(o2); {a:1, b:2, c:3}
    중괄호내의 값도 가져옴.
    유의사항
    var o1 ={a:1, b:2};
    var o2 = {a:2, ...o1}
    console.log(o2); {a:1, b:2} => 덮어 씌우기 됨.
    var o1 ={a:1, b:2};
    var o2 = {...o1, a:2}
    console.log(o2); { b:2, a:2} => 덮어 씌우기 됨.

    스프레드연산자는 중괄호, 소괄호, 대괄호에서만 사용할 수 있다.
