8. Map / Set 자료형 (ES6)

  • Map 자료형

    Map 자료형 만들기

    Map 자료형은 자료간의 연관성을 표현하기 위해 쓴다.

    Object 자료형은 자료이름으로 문자만 가능 / Map 자료형은 모든 데이터타입 가능

    var person = new Map();
    // person 객체에 키와 값 추가.
    person.set('name', 'Kim');
    person.set('age', '20');
    person.set(100, 'number');
    person.set([1,2,3], 'array');

    //출력하기
    person.get('name'); // Kim

    //삭제하기
    person.delete(100); // 100=>number 어트리뷰트 삭제

    // 데이터 개수
    person.size

    // 반복문으로 Map에서 자료 꺼내는 법
    for(var key of person.keys()){
    console.log(key); // [1,2,3] 100 age
    }

    Map 자료형에 직접 자료 넣기

    var person = new Map([
    ['name', 'Kim'],
    ['age', 20],
    [100, 'number']
    ])

  • Set 자료형

    Set 자료형 만들기

    var 출석부 = ['john', 'tom', 'andy', 'tom'];
    var 출석부2 = new Set(['john', 'tom', 'andy', 'tom']);
    // 중복을 허용하지 않고 같은 데이터는 한번만 넣는다.
    console.log(출석부2); // {'john', 'tom', 'andy'}

    Set 자료형은 중복을 허용하지 않는다.

    Set 자료형 자료 추가하기/제거하기/ 데이터 존재확인 / 자료개수

    //추가하기
    출석부2.add('sally');

    //제거하기
    출석부2.delete('tom');

    //데이터 존재 확인
    출석부2.has('sally');// true, false로 출력


    //데이터 개수
    출석부2.size;

  • Set 자료형의 활용

    데이터를 일반 배열에 넣을 경우, 모든요소가 같은게 있는지 확인하는 코드를 작성해야한다.

    배열의 중복자료를 제거하고 싶으면?

    var 출석부 = ['john', 'tom', 'andy', 'tom'];
    var 출석부2 = new Set(출석부);
    출석부 = [...출석부2];

    Set 자료형은 특히 배열의 데이터 중복을 제거할 때 많이 사용한다.
