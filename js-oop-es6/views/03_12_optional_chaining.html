11. ?. / ?? 연산자 (optional chaining)

  • 여러 문자, 숫자를 한 변수에 저장하려면

    ?.연산자

    중접된 object 자료에서 자료를 참조할 때 reference 에러 없이 안전하게 값을 가져올 수 있음.

    에러를 해결해주는 문법이 아니라 에러를 감춰주는 문법임을 유의하여 사용

    왼쪽의 변수가 undefined 또는 null이면 undefined를 반환

    var user = {   name: 'Mun',
      age: {value : 20},
    }

    console.log(user.age.value); //20

    // 없는 데이터를 호출(1depth) -> undefined
    console.log(user.country); // undefined
    // 없는 데이터를 호출(2depth) -> undefined.value를 호출한 것
    console.log(user.country.value);// undefined의 어트리뷰트는 존재하지 않으므로 참조에러가 뜬다.

    // ?. 연산자로 없는데이터를 호출하면
    console.log(user.country);// undefined
    console.log(user.country?.value);// undefined

    // null값도 동일하게 작동

  • ?? 연산자 (nullish coalescing 연산자)

    좌항이 undefined, null이면 우항의 값을 출력한다.

    var user;
    console.log(user ?? '로딩중');
