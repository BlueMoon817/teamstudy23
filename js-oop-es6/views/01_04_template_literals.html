3. 템플릿 리터럴

  • 백틱으로 문자열을 감싸면 문장 줄바꿈도 적용이 된다.

    문자열과 변수를 따로 + 연산자로 더하지 않고 같이 사용가능하다. 변수는 보간법( ${} )으로 감싸줌.

    var 백틱 = `안녕하세요 저는 ${ name }입니다.`;
    console.log(백틱); // 안녕하세요 저는 문유라입니다.

  • ES6-tagged literal

    함수를 호출할 때 그룹연산자 대신 문자를 붙여 실행시킬 수 있다.

    정규표현식으로 가능했던, 문자와 변수를 분리하여 사용가능하다.

    해체분석기 파라미터1 : 문자들을 Array에 담아줌

    해체분석기 파라미터2 : ${변수}를 뜻함

    해체분석기 파라미터3 : ${변수}를 뜻함
