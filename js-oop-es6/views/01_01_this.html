1. This의 의미

  • 자바스크립트엔진은 파싱할 때 window객체에 모든 내용을 담는다.
       이 때 전체코드를 감싸는 암묵적으로 {}(오브젝트)가 생성된다.

  • window는 최상위 객체이다.

  • 전역에서 선언한 모든 요소들은 window 객체의 어트리뷰트가 된다.

    함수는 전역에서 선언하면 window객체의 메서드가 된다.

  • this는 this가 표현된 식 또는 문을 감싸는 블록의 상위 객체를 가리킨다.

    • 전역에서의 this는 상위객체인 window 객체를 가리킨다.

    • window객체가 아닌 생성된 객체 내의 메서드에서의 this는,
            this가 표현된 식 또는 문을 감싸는 블록의 바로 상위 객체를 가리킨다.

    var 오브젝트 = {
      data : 'Kim',
      함수 : function(){
        console.log('안녕')
        // 오브젝트를 가리킴.
        console.log(this);
      }
    }
    오브젝트.함수();

  • 함수는 전역에서 선언하면 window객체의 메서드가 된다.

    익명함수는 전역 선언으로 여겨지기 때문에 window를 가리킨다.

    var arr = [1,2,3];
    arr.forEach(function(a){
      console.log(a); // 1,2,3
      console.log(this); // window객체
      // 콜백함수도 전역에서 사용된 일반함수이기 때문에
      // 이 함수 내에서의 this는 window를 가리킨다.
    });
    var 오브젝트 = {
      names: ['김', '이', '박'],
      func : function(){
        console.log(this); // 오브젝트
        // 콜백함수가 익명함수일경우
        오브젝트.names.forEach( function(){
          console.log(this); // window
        })
      }
    }

    콜백함수로 들어간 익명함수가 화살표함수일 경우 this가 표현된 문을 감싸는 상위의 this값을
            그대로 상속받는다.

    var 오브젝트 = {
      names: ['김', '이', '박'],
      func : function(){
        console.log(this); // 오브젝트
        // 콜백함수가 화살표함수일경우.
        오브젝트.names.forEach( () => {
          console.log(this);
          // 오브젝트
          // 오브젝트
          // 오브젝트
          // >> 상위의 this값을 상속받기 때문
        })
      }
    }

  • 이벤트리스너에서의 this는 e.currentTarget을 가리킨다. (이벤트가 발생한 노드객체)

    생성자 함수에서의 this는 생성될 객체를 가리킨다.

    var 어쩌구 = {}
    생성자함수
    function 기계 (){
      this.이름 = "Kim"
    }
    var 오브젝트 = new 기계();
    console.log(오브젝트.이름); // "Kim"
