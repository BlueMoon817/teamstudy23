  • 최댓값 구하기

    자바스크립트에서 최댓값을 구하고 싶을 때 Math.max()라는 기본 내장함수를 쓸 수 있다.

    ex. Math.max(5,6,4,3)

    var numbers = [2,3,4,5,6,1,3,2,5,5,4,6,7];

    numbers 안에 있는 숫자들을 Math.max()에 넣어서 쓰되 직접 하나하나 입력하지 않고 넣는 방법은?

    Math.max(...numbers);

  • 글자를 알파벳순으로 정렬해주는 함수

    자바스크립트는 array 내의 데이터를 알파벳순으로 정렬하고 싶을 때 sort()라는 array 내장함수를 사용할 수 있다.

    sort()는 배열에만 사용가능하다.

    //사용 예
    console.log( [ 'b', 'c', 'a' ].sort() );
    //[ 'a', 'b', 'c' ] 출력됨

    array가 아니라 문자열에도 적용할 수 있는 알파벳순 정렬함수 만들기

    정렬('bear') => 콘솔창에 알파벳 순으로 a b e r 출력시키기

    function 정렬(...rest){
    // 문자열은 sort적용이 불가능하므로 배열에 풀어 담아서 정렬한다.
    console.log([...rest].sort());
    }
    정렬('bear');
