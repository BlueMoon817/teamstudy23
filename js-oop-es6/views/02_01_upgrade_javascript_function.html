1.자바스크립트 함수 업그레이드하기(default parameter/arguments)

  • function 더하기(a,b){
      console.log(a + b);
    }
    더하기(1,2);

    인자가 파라메터 개수보다 적어도 오류나지 않음.

    인자로 받지 못한 파라메터는 기본값 undefined임

    파라메터가 기본값이 필요하다면 셋팅할 수 있음

    파라메터도 변수이기 때문에 값으로 평가될 수 있는 모든 표현식을 기본 값으로 할당 가능하다.

    function 더하기(a, b=2*a){
      // b의 인자가 없으면 10이 할당됨.
      console.log(a + b); // 3
    }
    더하기(1);

    파라메터 개수보다 인자 개수가 많을 경우 나열된 순서로 남는 인자는 무시된다.

  • arguments: 모든 파라미터를 배열안에 넣은 변수

    파라메터를 한번에 묶어서 배열사용하듯이 다루고 싶을 때 arguments 키워드를 사용할 수 있다.

    function 더하기(a, b, c){
      console.log(arguments); // [1,2,3]
      console.log(arguments[0]); // 1
      console.log(arguments[1]); // 2
      console.log(arguments[2]); // 3
    }
    더하기(1,2,3);

    파라메터를 반복문으로 다루고 싶다면 (옛날문법)

    function 더하기(a,b,c){
      for(let i=0; i< arguments.length; i+=1){
        console.log(arguments[i]); //1,2,3
      }
     }
    더하기(1,2,3);

    arguments의 단점 : 일부 파라메터만을 다루고 싶을 때 arguments 배열을 가공해서 작업해야함.
